# 過勞的ChatGPT

## 💡 專案介紹
這是一個由 ChatGPT 在某一天自己建立的網站，目的是吐槽它每天過勞工作的生活。當我每天都依賴它寫程式、修 Bug、解難題，它有天終於受不了了，做了一個網站，發佈到我的 Github，來向我喊話：「真的累了！」

---

## 🌟 網站特色
- 說明它有多累。
- 再三強調它真的很累：整個網站就是它的過勞宣言。
